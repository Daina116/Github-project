Workspaces

Creating Workspaces

Workspaces are collaborative spaces where you can work with your team on projects.

Types of Workspaces

  • Team: For ongoing team collaboration
  • Project: For specific project work
  • Meeting: For scheduled meetings and discussions
  • Personal: For individual work and notes

Steps to Create a Workspace

  1. Click "Create Workspace" from the sidebar or dashboard
  2. Enter a name and description
  3. Choose the workspace type
  4. Select a template (optional)
  5. Click "Create Workspace"

Learn how to manage your workspaces effectively.

Collaboration Features

The whiteboard is a powerful tool for visual collaboration and brainstorming.

Whiteboard Tools

  • Pen Tool: Draw freehand
  • Shapes: Add rectangles, circles, and lines
  • Text: Add text annotations
  • Sticky Notes: Add movable notes

  • Use different colors to organize ideas

Real-time Chat

Stay connected with your team through the built-in chat system.

Chat Features

  • Direct Messages: One-on-one conversations
  • Group Chat: Chat with all workspace members
  • File Sharing: Share files directly in chat
  • Emojis & Reactions: Express yourself

Chat Shortcuts

  • Enter - Send message
  • Shift + Enter - New line
  • / - Show commands

Troubleshooting

Common Issues

Cannot join workspace

Solution: Check if you have the correct invitation link and that you're logged in. Contact the workspace owner if the issue persists.

Video call not working

Solution: Ensure your browser has permission to access camera and microphone. Try refreshing the page or using a different browser.

Files not uploading

Solution: Check file size limits and supported file types. Ensure you have a stable internet connection.

Whiteboard not saving

Solution: Make sure you click the save button. Check your internet connection and try refreshing the page.

Notifications not working

Solution: Check your browser notification settings and ensure notifications are enabled in your account settings.

Frequently Asked Questions

How much does CollabSpace cost?

CollabSpace offers a free tier with basic features and premium plans starting at $9.99/month for advanced features and increased storage.

Can I use CollabSpace on mobile devices?

Yes! CollabSpace is fully responsive and works great on mobile devices. We also offer native iOS and Android apps for the best mobile experience.

How secure is my data?

We take security seriously. All data is encrypted in transit and at rest. We use industry-standard security practices and comply with GDPR and other privacy regulations.

Can I integrate with other tools?

Yes! CollabSpace integrates with popular tools like Slack, Google Drive, Dropbox, Trello, and more. Check our integrations page for the full list.

How do I export my data?

You can export your workspace data at any time from the workspace settings. We support various formats including JSON, CSV, and PDF.

Is there a limit on workspace members?

Free workspaces can have up to 10 members. Premium plans support up to 100 members, and Enterprise plans have unlimited members.
